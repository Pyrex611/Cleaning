Frequently Asked Questions

What is an ID CODE?

An ID CODE is a unique security authorization token required to process specific high-level transactions, such as transfers and withdrawals.

How do I purchase one?

Navigate to the Dashboard, tap on "Buy CODE", enter your full name, and complete the payment process via the available secure payment methods.

My transfer is pending. Why?

Transfers are usually instant. However, if an ID CODE is missing or invalid, the transaction will be placed on hold until authorized.
